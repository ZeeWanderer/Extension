# ``Extension``

Umbrella documentation for the Extension package. This target re-exports the module set below so you can import once and opt into all extensions.

## Overview

Extension is a layered collection of modules that extend the Swift standard library, Foundation, CoreGraphics, UIKit, SwiftUI, SpriteKit, Observation, os logging, Accelerate, and Swift macros. Use the module pages to navigate by framework or feature area and keep dependencies explicit when you only need a subset.
