# ``CoreGraphicsExtension``

CoreGraphics-centric numerics, protocols, and geometry helpers.

## Overview

CoreGraphicsExtension defines 2D numeric protocols and adds convenience APIs to CoreGraphics types. It includes interpolation helpers, clamping utilities, and small geometry conveniences to keep math-heavy UI code terse and predictable.

## Topics

### 2D Numeric Protocols

- ``Numeric2D``
- ``SignedNumeric2D``
- ``AdditiveArithmetic2D``
- ``Equatable2D``
- ``Hashable2D``
- ``FloatingPoint2D``
- ``Uniform2D``

### Math and Geometry

- ``lerp(_:min:max:)->CGRect``
- ``lerp(_:min:max:)-6esqb``
- ``ilerp(_:min:max:)-53q91``
- ``union(_:)``
